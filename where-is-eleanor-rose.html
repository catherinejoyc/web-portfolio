The Project

Where is Eleanor Rose? is a 2D top-down adventure game, that addresses the 5 stages of grief and takes place inside of the mind of a teenage schoolgirl.

My Work

As the only programmer I coded every piece of gameplay behaviour - except for the pathfinding system for which I used this A* plugin.
An essential aspect in game development is to code developer friendly so I made sure that the artists, audio designer and level designer were able to drag their content into the unity inspector without ever having to deal with the source code.
