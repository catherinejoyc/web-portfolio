The Project

This application explains the user the voting process for Austrian elections (for example the Gemeinderatswahl 2020). Its purpose is to educate and motivate people to vote, since many - especially young people - do not know how to vote.

Since this project is still in its planning stage there isn't much to show but I will update this page as soon as possible. If you are interested in this project and learning more about it you can message the project leader Sebastian Pokorny.
